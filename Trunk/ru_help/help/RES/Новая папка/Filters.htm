Справочная система программы Nevod AntiSpam

Работа с фильтрами

Общие сведения

В Nevod AntiSpam существует две группы фильтров:

  • Белый список. Сообщение, которое соответствует одному из правил этого списка, пропускается без изменений.
  • Черный список. Сообщение, которое соответсвует одному их правил этого списка, помечается как спам ( его тема заменяется на It's spam:<Reason>, где Reason - причина, по которой сообщение было помечено).

Каждое почтовое сообщение перед отправкой локальному клиенту анализируется в таком порядке:

  • Сообщение анализируется фильтрами, входящими в Группу разрешения; если результат положительный, письмо пропускается без изменений.
  • Если же анализ фильтрами Группы разрешения не дал результатов, производится анализ фильтрами, входящими в Группу запрета. В случае положительного результата сообщение помечается как спам, иначе - пропускается без изменений.

 

В Белый список входят следующие фильтры

  1. Отправители - список адресов адресов отправителей, которым разрешено отправлять почту. В адресах разрешается использовать символы-заменители *(произвольное количество произвольных символов) и ? (один произвольный символ). Например, если создано правило для j*smoth@example.com, почта будет приниматься от jsmith@examle.com, johnsmith@example.com и тому подобное; для правила ?smith@example.com это будут jsmith@example.com, xsmith@example.com и тому подобные.
  2. Слова - Если слово, присутствующее в списке, найдено в почтовом сообщении, то данное сообщение будет пропущено. Для каждого слова можно указать место, где производить его поиск(в теме, тексте или в теме и в тексте сообщения) .
  3. Приложенный файлы - Если в почтовом сообшении есть приложенный файл, и расширение этого файла есть в списке, сообщение будет пропущено.
  4. Штампы.
    В свое почтовое сообщение отправитель может добавить Nevod Stamp. Nevod Stamp представляет собой строку в виде <Nevod AntiSpam :"keyword">, где keyword - произвольное слово (оно также должно присутствовать в списке). Если при анализе соообщения будет найден Nevod Stamp и ключевое слово будет присутствовать в списке, сообщение будет пропущено.

В Черный список входят следующие фильтры

  1. Отправители - список адресов адресов отправителей, которым разрешено отправлять почту. В адресах разрешается использовать символы-заменители *(произвольное количество произвольных символов) и ? (один произвольный символ).Например, если создано правило для j*smoth@example.com, почта от jsmith@examle.com, johnsmith@example.com и тому подобное будет помечаться как спам; для правила ?smith@example.com это будут jsmith@example.com, xsmith@example.com и тому подобные.
  2. Слова - Если слово, присутствующее в списке, найдено в почтовом сообщении, то данное сообщение будет помечено как спам. Для каждого слова можно указать место, где производить его поиск(в теме, тексте или в теме и в тексте сообщения) .
  3. Приложенный файлы - Если в почтовом сообшении есть приложенный файл, и расширение этого файла есть в списке, сообщение будет помечено как спам.
  4. Максимальное количество изображений - производится подсчет количества изображений в сообщении, и если их количество превышает заданное значение, сообщение будет помечено как спам.
  5. Максимальное количество ссылок в сообщении - производится подсчет количества ссылок в сообщении, и если их количество превышает заданное значение, сообщение будет помечено как спам.
  6. Максимальный размер сообщения - если размер сообщения превышает заданное значение, сообщение помечается как спам.

Кажое правило фильтра может находится в одном их двух состояний - включенном (правило используется при анализе сообщений) и выключенном (правило не используется при анализе сообщений).
