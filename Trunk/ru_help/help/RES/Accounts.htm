Справочная система программы Nevod AntiSpam

Учетные записи

Добавление

1. В окне добавления учетной записи (Учетные записи| Добавить) введите следующие данные:
  • Имя учетной записи - произвольное уникальное имя для добавляемой учетной записи
  • Имя пользователя - имя учетной записи на удаленном почтовом сервере
  • Пароль - пароль учетной записи на удаленном почтовом сервере
  • Почтовый сервер - имя удаленного почтового сервера
  • Порт - порт, на котором работает удаленный почтовый сервер (по умолчанию - 110)
  • Таймаут - время(сек), в течении которого будет производится попытка установить подключение с удаленным почтовым сервером

Для проверки введенных параметров, нажмите на кнопку Проверить.
2.Запуститите ваш почтовый клиент (Outlook Express, Eudora,Mozilla Thunderbird) и зайдите в настройки соответствующей учетной записи. Измените имя сервера входящей почты (POP3-сервера) на localhost,номер порта - на 1000 (или тот, который вы установили в настройках Nevod AntiSpam) , имя пользователя - на имя соответсвтующей учетной запими в Nevod AntiSpam.
3. Создайте правило сортировки по теме сообщения "It's spam". Все сообщения, распознанные как спам, будут автоматически попадать в эту папку.

Редактирование

Выделите необходимую учетную запись и в контекстном меню выберите пункт Редактировать.

Удаление

Выделите необходимую учетную запись и в контекстном меню выберите команду Удалить.

Получение новых сообщений

1. Выделите необходимую учетную запись и в контекстном меню выберите пункт Проверить.
2. Для всех учетных записей - в контекстном меню выберите пункт Проверить все.

Если включен режим регулярной проверки (Учетные записи| Проверять регулярно),то учетные записи будут проверяться периодически через заданный промежуток времени.

Прекращение получния новых соообщений

1. Выберите необходимую учетную запись и в контекстном меню выберите пункт Остановить.
2. Для всех учетных записей - в контекстном меню выберите пункт Остановить все.
