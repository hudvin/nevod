Справочная система программы Nevod AntiSpam

Настройки

Общие настройки

  • Загружать при загрузке -если флажок установлен, программа будет загружаться при загрузке системы
  • Порт сервера - порт, на котором будет работать локальный почтовый сервер. Внимание ! Если вы установите порт, отличный от стандартного, не забудьте произвести соответствующие изменения во всех учетных записях вашего почтового клиента.
  • Проверять каждые ... минут - промежуток времени, через который будет производится проверка учетных записей на наличие новых сообщений.
  • Проверять наличие новых сообщений если подключение к сети не обнаружено -если флажок установлен, проверка на наличие новых сообщений производится только в том случае, если обнаружено подключение к Интернет.
  • Фильтровать сообщения - если флажок установлен, антиспам фильтр будет активирован.

Уведомления

  • При получении новой почты -если флажок установлен, при получении новой почты будет отображено всплывающее сообщение.
  • При ошибке - если флажок установлен и в процессе получения новой почты произойдет ошибка, будет отображено всплывающее сообщение.

Звуки

  • При получении новой почты -если флажок установлен, при получении новой почты будет проиграна мелодия. Мелодию можно выбрать, нажав на кнопку Browse. Для прослушивания мелодии нажмите на кнопку Play.
  • При ошибке -если флажок установлен, и в процессе получения новой почты произойдет ошибка. Мелодию можно выбрать, нажав на кнопку Browse.Для прослушивания мелодии нажмите на кнопку Play.

Слежение за буфером обмена

Технология слежение за буфером обмена позволяет очень быстро и очень просто производить добавление адресов URL и Email в белый или черный список. Достаточно всего лишь выделить нужный адрес URL или Emailи скопировать его в буфер обмена - и адрес автоматически будет добавлен в Белый(или Черный) список!

  • Следить за буфером обмена -если флажок установлен,то слежение за буфером обмена будет включено..
  • Показывать Редактор перед добавлением - если флажок установлен, топри обнаружении адреса будет выведено окно Редактора, в котором вы можете изменить параметры добавляемого адреса.
  • Добавлять в Белый список/Черный список - Определяет, в какой список будет автоматически добавлен адрес.
  • Следить за
    • URL - будут отслеживаться только адреса URL
    • Email - будут отслеживаться только адреса Email
    • URL и EMail - будут отслеживаться адреса URL и Email

Горячие клавиши

Все горячие клавиши являются глобальными - даже если вы работает в другом приложении,то,нажав горячую комбинацию, вы получите доступ к нужной функции.
Для назначения комбинации клавиш, отличной от стандартной, установите курсор в соответсвующем поле и нажмите желаемую комбинацию, а затем кнопку Назначить.Если назначение прошло успешно, в поле ввода появится новая комбинация, в противном случае комбинация останется прежней.

  • Добавить новый фильтр - При нажатии соотв. комбинации будет показан Редактор для добавления нового правила фильтрования.
  • Показать окно программы - При нажатии соотв. комбинации будет показано главное окно Nevod AntiSpam
  • Проверить все учетный записи - При нажатии соотв. комбинации будет запущена проверка всех учетных записей.
  • Запустить почтовый клиет - При нажатии соотв. комбинации будет запущена ваша почтовая программа.
