Справочная система программы Nevod AntiSpam

Учетные записи

Добавление

1. В окне добавления учетной записи (Учетные записи| Добавить) введите следующие данные:

  • Имя учетной записи - произвольное уникальное имя для добавляемой учетной записи
  • Имя пользователя - имя учетной записи на удаленном почтовом сервере
  • Пароль - пароль учетной записи на удаленном почтовом сервере
  • Почтовый сервер - имя удаленного почтового сервера
  • Порт - порт, на котором работает удаленный почтовый сервер (по умолчанию - 110)
  • Таймаут - время, в течении которого будет производится попытка установить подключение с удаленным почтовым сервером

Для проверки введенных параметров, нажмите на кнопку Проверить.

2.Запуститите ваш почтовый клиент (Outlook Express, Eudora,Mozilla Thunderbird) и зайдите в настройки соответствующей учетной записи. Измените имя сервера входящей почты (POP3-сервера) на localhost, порт - на 1000 (или тот, который вы установили в настройках Nevod AntiSpam) , имя пользователя - на имя соответсвтующей учетной запими в Nevod AntiSSpam.

3. Создайте правило сортировки по теме сообщения "It's spam". Все сообщения, распознанные как спам, будут автоматически попадать в эту папку.

Редактирование

Для редактирования параметров учетной записи необходимо выделить ее и затем сделайте на ней двойной щелчок мышью.

Удаление

Выделите учетную запись, которую вы хотите удалить, нажмите на ней правой кнопкой мыши и выберите команду Удалить

Получение новых сообщений

1. Для выделенной учетной записи - нажмите правой кнопкой мыши и выберите пункт Проверить

2. Для всех учетных записей - в контекстном меню выберите пункт проверить все.

Если включен режим регулярной проверки (Учетные записи| Проверять регулярно),то новые сообщения будут загружаться через заданный промежуток времени.

Прекращение получния новых соообщений

1. Для выделенной учетной записи - нажмите правой кнопкой мыши и выберите пункт Остановить

2. Для всех учетных записей - в контекстном меню выберите пункт Остановить все..
