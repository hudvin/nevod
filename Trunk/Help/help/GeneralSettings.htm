Nevod AntiSpam

Settings

General Settings

  • Run on startup -when this option is checked, Nevod AntiSpam will be automatically loaded at Windows startup.
  • Local port - port, on which will work local mail server.
  • Check every ... minutes - time interval, through which there will be it is produced checking accounts to the presence of new messages..
  • Check the presence of new communications if connection to the network is not discovered-when this option is checked, checking to the presence of new messages it is produced only in such a case, when discovered connection to the Internet.
  • Enable filtering - -when this option is checked, antispam filter will be activated.

Messages

  • When new messages are loaded - when this option is checked, the message will be shown at loading new messages.
  • When errors occured -when this option is checked, the message will be shown if error occured.

Sounds

  • When new messages are loaded - when this option is checked, the sound will be played at loading new messages.
  • When errors occured - when this option is checked, the sound will be played if error occured.

Tracking for the clipboard

Tracking for the clipboard technology allows very rapidly and very simple to produce the addition of addresses URL and Email to the white or black list. It suffices altogether to only select the necessary address URL or Emaili to copy to the clipboard - and address will be automatically added in White (or Black) )list!

  • Enable clipboard tracking - when this option is checked, tracking for the clipboard will be activated.
  • Показывать Редактор перед добавлением - если флажок установлен, топри обнаружении адреса будет выведено окно Редактора, в котором вы можете изменить параметры добавляемого адреса.
  • Добавлять в Белый список/Черный список - Определяет, в какой список будет автоматически добавлен адрес.
  • Следить за
    • URL - будут отслеживаться только адреса URL
    • Email - будут отслеживаться только адреса Email
    • URL и EMail - будут отслеживаться адреса URL и Email

Hotkeys

Все горячие клавиши являются глобальными - даже если вы работает в другом приложении,то,нажав горячую комбинацию, вы получите доступ к нужной функции.
Для назначения комбинации клавиш, отличной от стандартной, установите курсор в соответсвующем поле и нажмите желаемую комбинацию, а затем кнопку Назначить.Если назначение прошло успешно, в поле ввода появится новая комбинация, в противном случае комбинация останется прежней.

  • Добавить новый фильтр - При нажатии соотв. комбинации будет показан Редактор для добавления нового правила фильтрования.
  • Показать окно программы - При нажатии соотв. комбинации будет показано главное окно Nevod AntiSpam
  • Проверить все учетный записи - При нажатии соотв. комбинации будет запущена проверка всех учетных записей.
  • Запустить почтовый клиет - При нажатии соотв. комбинации будет запущена ваша почтовая программа.
